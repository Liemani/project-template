# project-template

- `raw/`
  - `inbox/`: 아직 분류하지 않은 유입 원문 파일
  - `original/`: 최종 보관할 원본 파일
  - `assets/`: 이미지 등 자산 파일
- `project-name/`
  - 실제 프로젝트 생성 시 프로젝트명으로 치환되는 monorepo 루트
  - `README.md`: 작업 디렉터리 구조/용도 안내
  - `extracted/`: 원문에서 그대로 추출한 markdown/text 산출물
  - `wiki/`: 사람용 정리 문서와 지식 문서
  - `requirements/`: 요구사항 단위 문서와 추적성 관리
  - `tasks/`: 구현 진행을 위한 작업 단위 관리
  - `decisions/`: 구현 방식과 설계 판단 기록
